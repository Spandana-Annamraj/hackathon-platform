# Online Hackathon Application & Evaluation Platform

A full-stack web platform designed to manage online hackathons efficiently. The platform allows participants to register teams, submit projects, judges to evaluate submissions, and admins to control the hackathon workflow.

---

## Features

### Participant Module

* User Signup & Login
* Team Registration
* Project Submission
* Dashboard Access

### Judge Module

* View Submitted Projects
* Evaluate Projects
* Give Scores & Feedback
* View Evaluation Summary

### Admin Module

* Control Submission Window
* Manage Platform Access
* Monitor Project Submissions

### Additional Features

* Leaderboard System
* JWT Authentication
* Protected Routes
* MongoDB Database Integration
* Responsive Frontend

---

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Axios
* React Router DOM

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Installation & Setup

### Clone Repository

```bash
git clone https://github.com/Spandana-Annamraj/hackathon-platform.git
cd hackathon-platform
```

---

## Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file inside the backend folder:

```env
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```

Run backend server:

```bash
npm start
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```
